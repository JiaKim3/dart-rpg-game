# 🧙 Dart 전투 RPG 게임

## 📌 과제 개요
- Dart의 객체지향, 파일 입출력, 랜덤 기능을 활용하여 전투 RPG 게임을 구현하였습니다.
- 플레이어는 캐릭터로서 몬스터들과 차례로 대결하며 승리 또는 패배에 도달합니다.

---

## 🚀 실행 방법

### ✅ 준비 파일
- `main.dart` : 게임 실행 메인 코드
- `characters.txt` : 캐릭터의 스탯 파일 (예: `50,10,5`)
- `monsters.txt` : 몬스터 스탯 리스트 (예: `Batman,30,20` 등)

### ▶ 실행 명령어

```bash
dart run main.dart
